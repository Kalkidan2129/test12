
# Power BI Dashboard with Healthcare Data

## Project Overview

This project involved building a comprehensive Power BI dashboard using healthcare insurance data sourced from SQL Server databases. The dashboard features custom measures, slicers, and various chart types to deliver actionable insights for strategic planning. The solution emphasizes effective data storytelling through well-designed visualizations aligned with business requirements.

---

## Business Problem

Healthcare insurance data is rapidly expanding, making it challenging to extract timely and relevant insights for strategic business planning. Traditional reporting methods lack flexibility and do not support interactive exploration of key metrics. There is a need for a self-service BI solution that enables quick discovery and visualization of critical healthcare data.

---

## Objective

- Connect and import healthcare insurance data from SQL Server into Power BI.
- Create meaningful metrics and measures using DAX for accurate data aggregation.
- Design interactive and customizable visualizations to support strategic decision making.

---

## Tools & Technologies

- Power BI Desktop
- SQL Server Management Studio
- SQL Server
- DAX (Data Analysis Expressions)
- Power BI Service
- Microsoft Windows OS

---

## Project Workflow

- Connect to SQL Server and execute scripts to create the data mart.
- Import selected tables from SQL Server into Power BI Desktop.
- Create custom measures using DAX to calculate key metrics.
- Build interactive visualizations including slicers, cards, pie charts, and stacked bar charts.
- Finalize dashboard layout with branding elements and apply color themes.

---

## Key Insights

- Distinct count of members provides a core metric for membership analysis.
- Pie charts effectively visualize high-level categorical data such as sales by region or product.
- Stacked bar charts enable comparison of grouped values across multiple dimensions.
- Cards highlight single critical values, enhancing dashboard clarity for decision makers.


# Power BI Dashboard with Healthcare Data

## Project Overview

This project involved building a comprehensive Power BI dashboard using healthcare insurance data sourced from SQL Server databases. The dashboard features custom measures, slicers, and various visualizations designed to provide actionable insights for strategic planning. The step-by-step tutorial guides users through data import, metric creation, and visualization best practices.

---

## Business Problem

Healthcare insurance data is rapidly expanding, making it challenging for organizations to quickly extract meaningful insights for strategic business planning. Traditional reporting methods often lack flexibility and interactivity, limiting the ability to explore data effectively. There was a need for a self-service BI solution that enables users to create customizable and insightful visualizations efficiently.

---

## Objective

- Connect and import healthcare insurance data from SQL Server into Power BI.
- Create custom metrics and measures using DAX to analyze membership data.
- Design interactive and visually appealing dashboards to support strategic decision making.

---

## Tools & Technologies

- Power BI Desktop
- SQL Server
- SQL Server Management Studio
- Data Analysis Expressions (DAX)
- Power BI Service
- Microsoft Windows OS

---

## Project Workflow

- Connect to SQL Server and execute scripts to create data marts.
- Import required tables into Power BI Desktop using Get Data feature.
- Create custom measures and calculated fields with DAX.
- Build interactive visualizations including slicers, cards, pie charts, and stacked bar charts.
- Finalize dashboard layout with images, themes, and company branding.

---

## Key Insights

- Power BI supports seamless integration with SQL Server for efficient data import.
- Custom DAX measures enable precise aggregation and analysis of healthcare membership data.
- Interactive slicers and visualizations enhance user-driven data exploration.
- Effective dashboard design focuses on storytelling by highlighting key business metrics.
